PM External Falsification Checklist
1. Light‑Propagation Tests (Optical Sector)

These probe the n=eϕ refractive‑gravity mechanism directly.

    Strong‑field light bending (EHT, Sgr A, M87)

    Weak‑lensing shear profiles (galaxy clusters, cosmic shear)

    Strong‑lensing arcs and Einstein radii

    Shapiro delay (Cassini, pulsars)

    Frequency‑dependent propagation (FRBs, pulsar timing)

    Photon‑sphere / shadow size (EHT constraints)

Status: You’ve addressed some of the weak‑field behaviour; strong‑field bending is still open.
2. Orbital Dynamics (Massive Sector)

These test the corrected force law a=c22∇ϕ.

    Perihelion precession (Mercury, binary pulsars)

    Binary pulsar orbital decay (Hulse–Taylor, double pulsar)

    Frame dragging / Lense–Thirring (Gravity Probe B, LAGEOS)

    Gravitational redshift (Pound–Rebka, GPS clocks)

    Time dilation in gravitational wells (atomic clocks at altitude)

Status: Newtonian limit is now correct; relativistic corrections still need checking.
3. Compact‑Object Structure

These test the PM EOS, density law, and absence of horizons.

    Mass–radius curve (NICER, X‑ray bursts)

    Maximum mass (LIGO/Virgo/KAGRA neutron‑star mergers)

    Surface redshift (spectral lines from neutron stars)

    Post‑merger gravitational‑wave spectrum

    Absence of event horizons (EHT brightness inside shadow)

Status: You’ve built the PM structure equations; observational comparison is still ahead.
4. Gravitational Waves

These probe the medium’s dynamics and the optical metric’s effect on wave propagation.

    Propagation speed (GW170817 + GRB170817A)

    Dispersion (frequency‑dependent arrival times)

    Waveform phasing (inspiral templates)

    Ringdown spectrum (QNMs vs PM surface modes)

    Echoes or lack thereof (post‑merger signatures)

Status: Not yet tested in PM.
5. Cosmology

These test the density law ρ=ρnuceϕ and the PM Poisson equation on large scales.

    Growth rate of structure fσ8(z)

    Matter power spectrum (BAO, galaxy surveys)

    CMB lensing amplitude

    Integrated Sachs–Wolfe effect

    Large‑scale gravitational potentials (ISW cross‑correlations)

Status: Untouched so far — but extremely constraining.
6. High‑Precision Local Tests

These probe tiny deviations from Newtonian gravity.

    Inverse‑square‑law tests (torsion balances)

    Equivalence principle tests (MICROSCOPE)

    Local gravitational constant variation

    Laboratory refractive‑index constraints (if PM predicts any EM coupling)

Status: Some of these are likely already tight enough to constrain PM’s couplings.
