\section*{Overview}

This document presents the unified Pushing--Medium (PM) field theory in a
gravity-first structure. Part I contains the complete, validated gravitational
sector (PM Gravity v2). Part II extends the same PM-medium fields to include
electrostatics, magnetostatics, and plasma contributions through unified source
terms.

The PM medium is described by two dynamical fields:


\[
\phi(\mathbf r,t) = \ln n(\mathbf r,t),
\qquad
\mathbf u(\mathbf r,t) = \text{medium flow field}.
\]



The scalar field $\phi$ determines the refractive index $n$, while the vector
field $\mathbf u$ encodes flow, advection, and moving-source effects.
